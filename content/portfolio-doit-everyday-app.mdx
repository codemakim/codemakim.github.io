# 안드로이드 앱 - 매일두잇!

## 📱 프로젝트 소개

주기적으로 해야하는 일을 기록하고 얼마나 잘 수행하고 있는지 확인하는 습관 관리 앱입니다. 모든 정보를 휴대폰에 저장하므로 **인터넷 연결 없이** 사용할 수 있습니다.

### 📋 프로젝트 정보

- **프로젝트 구분**: 개인 프로젝트
- **수행 기간**: 2019-09-27 ~ 2019-10-30 (약 1개월)
- **플랫폼**: Android
- **상태**: ⚠️ 현재 업데이트되지 않아 사용 불가

## 🔧 기술 스택

### Frontend

- **Dart**: Google에서 개발한 프로그래밍 언어
- **Flutter**: 크로스 플랫폼 모바일 앱 개발 프레임워크

### Database

- **SQLite (sqflite)**: 로컬 데이터베이스로 오프라인 환경에서 데이터 관리

## ✨ 주요 기능

### 1. 습관 등록 및 관리

- 반복적으로 수행해야 할 일들을 등록
- 카테고리별 분류 가능
- 수행 주기 설정 (매일, 주간, 월간 등)

### 2. 진행 상황 추적

- 일별 완료 여부 체크
- 연속 달성 일수 표시
- 달성률 통계 제공

### 3. 오프라인 지원

- 모든 데이터를 로컬에 저장
- 인터넷 연결 없이 완전한 기능 사용 가능

### 4. 직관적인 UI/UX

- Material Design 적용
- 간편한 체크 인터페이스
- 시각적 진행 상황 표시
