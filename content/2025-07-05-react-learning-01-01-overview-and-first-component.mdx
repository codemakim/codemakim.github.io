# React 학습 가이드 - 개요 & 첫 번째 컴포넌트

## 개요

React는 사용자 인터페이스를 렌더링하는 JavaScript 라이브러리다.

핵심 개념:

- UI는 버튼, 텍스트, 이미지 같은 작은 단위들로 구성된다
- React는 이들을 재사용 가능하고 중첩 가능한 컴포넌트로 결합한다
- 웹사이트부터 모바일 앱까지 모든 화면을 컴포넌트로 분해할 수 있다

### 이 챕터에서 배울 내용

- 첫 React 컴포넌트 작성하기
- 멀티 컴포넌트 파일 생성 방법
- JSX로 JavaScript에 마크업 추가하기
- JSX에서 중괄호로 JavaScript 기능 사용하기
- Props로 컴포넌트 설정하기
- 조건부 렌더링
- 여러 컴포넌트 한 번에 렌더링하기
- 순수 컴포넌트로 버그 방지하기
- UI를 트리로 이해하기

## 첫 번째 컴포넌트

### 컴포넌트란?

React 애플리케이션은 `컴포넌트`라는 독립적인 UI 조각으로 구성된다.

핵심 특징:

- React 컴포넌트는 마크업을 포함할 수 있는 JavaScript 함수다
- 버튼만큼 작을 수도, 전체 페이지만큼 클 수도 있다

### 예제: Gallery 컴포넌트

하나의 컴포넌트를 여러 번 재사용하는 방법이다.

```jsx
// Profile 컴포넌트 정의
function Profile() {
  return <img src="https://i.imgur.com/MK3eW3As.jpg" alt="Katherine Johnson" />;
}

// Gallery 컴포넌트: Profile을 3번 재사용
export default function Gallery() {
  return (
    <section>
      <h1>Amazing scientists</h1>
      <Profile />
      <Profile />
      <Profile />
    </section>
  );
}
```

코드 설명:

1. Profile 컴포넌트: 과학자의 이미지를 보여주는 간단한 컴포넌트다.
   - `function Profile()`로 컴포넌트를 정의한다.
   - JSX로 `<img>` 태그를 반환한다.

2. Gallery 컴포넌트: Profile을 3번 사용하는 갤러리다.
   - `export default`를 사용해 이 컴포넌트를 다른 파일에서 가져다 쓸 수 있게 한다.
   - `<Profile />`을 3번 작성하면 같은 컴포넌트가 3번 렌더링된다.
   - 일반 HTML처럼 `<section>`, `<h1>` 태그를 함께 사용할 수 있다.

결과:

- 화면에 "Amazing scientists" 제목과 함께 동일한 이미지가 3개 표시된다.
- 각 `Profile`은 독립적으로 렌더링되지만, 같은 코드를 재사용한다.

### 핵심 포인트

1. 함수 컴포넌트: `function ComponentName() { }` 형태로 정의한다
2. JSX 반환: HTML처럼 보이는 마크업을 반환한다
3. 재사용성: 같은 컴포넌트를 여러 번 사용할 수 있다
4. 중첩 가능: 컴포넌트 안에 다른 컴포넌트를 포함할 수 있다

### Vue와 비교

```html
<!-- Vue: Single File Component -->
<template>
  <section>
    <h1>Amazing scientists</h1>
    <Profile />
    <Profile />
    <Profile />
  </section>
</template>

<script setup>
  import Profile from "./Profile.vue";
</script>
```

```jsx
// React: 함수 컴포넌트
export default function Gallery() {
  return (
    <section>
      <h1>Amazing scientists</h1>
      <Profile />
      <Profile />
      <Profile />
    </section>
  );
}
```

차이점:

- Vue: `<template>` 태그로 마크업을 분리한다
- React: JSX를 함수에서 직접 반환한다
