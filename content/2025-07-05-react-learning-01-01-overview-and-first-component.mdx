# React 학습 가이드 - 개요 & 첫 번째 컴포넌트

## 개요

**React란?**  
React는 사용자 인터페이스(UI)를 렌더링하는 JavaScript 라이브러리입니다.

**핵심 개념:**

- UI는 버튼, 텍스트, 이미지 같은 작은 단위들로 만들어져
- React는 이걸 **재사용 가능하고 중첩 가능한 컴포넌트**로 묶어줘
- 웹사이트든 모바일 앱이든 모든 화면을 컴포넌트로 쪼갤 수 있어

### 이 챕터에서 배울 내용

- 첫 React 컴포넌트 작성하기
- 멀티 컴포넌트 파일 생성 방법
- JSX로 JavaScript에 마크업 추가하기
- JSX에서 중괄호로 JavaScript 기능 사용하기
- Props로 컴포넌트 설정하기
- 조건부 렌더링
- 여러 컴포넌트 한 번에 렌더링하기
- 순수 컴포넌트로 버그 방지하기
- UI를 트리로 이해하기

## 첫 번째 컴포넌트

### 컴포넌트란?

React 앱은 **컴포넌트**라는 독립적인 UI 조각들로 이루어져 있어.

**핵심 특징:**

- React 컴포넌트는 **마크업을 포함할 수 있는 JavaScript 함수**야
- 버튼처럼 작을 수도 있고, 페이지 전체만큼 클 수도 있어

### 예제: Gallery 컴포넌트

아래 예제는 하나의 컴포넌트를 여러 번 재사용하는 방법을 보여줍니다.

```jsx
// Profile 컴포넌트 정의
function Profile() {
  return (
    <img
      src="https://i.imgur.com/MK3eW3As.jpg"
      alt="Katherine Johnson"
    />
  );
}

// Gallery 컴포넌트: Profile을 3번 재사용
export default function Gallery() {
  return (
    <section>
      <h1>Amazing scientists</h1>
      <Profile />
      <Profile />
      <Profile />
    </section>
  );
}
```

**코드 설명:**

1. **Profile 컴포넌트**: 과학자 이미지를 보여주는 간단한 컴포넌트야.
   - `function Profile()`로 컴포넌트를 만들어.
   - JSX로 `<img>` 태그를 리턴해.

2. **Gallery 컴포넌트**: Profile을 3번 쓰는 갤러리야.
   - `export default`를 붙이면 다른 파일에서도 이 컴포넌트를 가져다 쓸 수 있어.
   - `<Profile />`을 3번 쓰면 똑같은 컴포넌트가 3번 렌더링돼.
   - HTML처럼 `<section>`, `<h1>` 태그랑 섞어서 쓸 수 있어.

**결과:**

- 화면에 "Amazing scientists" 제목이랑 같은 이미지가 3개 나와.
- 각 `Profile`은 따로따로 렌더링되지만, 코드는 재사용하는 거지.

### 핵심 포인트

1. **함수 컴포넌트**: `function ComponentName() { }` 이렇게 만들어
2. **JSX 반환**: HTML처럼 생긴 마크업을 리턴해
3. **재사용성**: 같은 컴포넌트를 여러 번 갖다 쓸 수 있어
4. **중첩 가능**: 컴포넌트 안에 다른 컴포넌트를 넣을 수 있어

### Vue와 비교

```html
<!-- Vue: Single File Component -->
<template>
  <section>
    <h1>Amazing scientists</h1>
    <Profile />
    <Profile />
    <Profile />
  </section>
</template>

<script setup>
import Profile from './Profile.vue'
</script>
```

```jsx
// React: 함수 컴포넌트
export default function Gallery() {
  return (
    <section>
      <h1>Amazing scientists</h1>
      <Profile />
      <Profile />
      <Profile />
    </section>
  );
}
```

**차이점:**

- Vue: `<template>` 태그로 마크업을 분리해서 써
- React: JSX를 함수에서 바로 리턴해
