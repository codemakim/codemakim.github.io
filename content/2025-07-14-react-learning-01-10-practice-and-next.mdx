# React 학습 가이드 - 연습문제 & 다음 단계

> React 공식 문서를 기반으로 한 학습 자료  
> 원문: [Describing the UI - React](https://react.dev/learn/describing-the-ui)

---

## 이 챕터에서 배운 내용 요약

✅ **첫 번째 컴포넌트**: 함수로 작성, JSX 반환  
✅ **Import/Export**: 파일 분리로 코드 구조화  
✅ **JSX**: JavaScript에 HTML 마크업 작성  
✅ **중괄호 `{}`**: JSX에서 JavaScript 표현식 사용  
✅ **Props**: 부모 → 자식 데이터 전달  
✅ **조건부 렌더링**: `if`, `&&`, `? :`로 조건 처리  
✅ **리스트 렌더링**: `map()`으로 배열 → JSX 배열 변환, `key` 필수  
✅ **순수 컴포넌트**: 같은 입력 → 같은 출력, 부작용 없음  
✅ **UI 트리**: 렌더 트리와 모듈 의존성 트리 이해

## 해답 예시

### 1. 프로필 카드 해답

```jsx
function ProfileCard({ user }) {
  return (
    <div className="profile-card">
      <img src={user.avatar} alt={user.name} />
      <h3>{user.name}</h3>
      <p>{user.role}</p>
    </div>
  );
}

export default function App() {
  return (
    <div className="profiles">
      {users.map(user => (
        <ProfileCard key={user.id} user={user} />
      ))}
    </div>
  );
}
```

### 2. 할 일 목록 해답

```jsx
function TodoItem({ todo }) {
  return (
    <li className="todo-item">
      {todo.completed ? (
        <del>{todo.text} ✅</del>
      ) : (
        todo.text
      )}
    </li>
  );
}

export default function TodoList() {
  return (
    <ul>
      {todos.map(todo => (
        <TodoItem key={todo.id} todo={todo} />
      ))}
    </ul>
  );
}
```

### 3. 상품 필터링 해답

```jsx
function ProductList({ category }) {
  const filteredProducts = products.filter(
    product => product.category === category
  );
  
  return (
    <ul>
      {filteredProducts.map(product => (
        <li key={product.id}>
          {product.name} - ${product.price}
        </li>
      ))}
    </ul>
  );
}
```
