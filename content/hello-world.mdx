# 안녕하라! 👋

이것은 제 첫 번째 블로그 포스트이다. Next.js와 contentlayer를 사용해서 만든 블로그에 오신 것을 환영한다!

## 기술 스택

이 블로그는 다음 기술들로 만들어졌다:

- Next.js 15: React 기반의 풀스택 프레임워크
- Contentlayer: MDX 파일을 타입 안전하게 관리
- Tailwind CSS: 유틸리티 우선 CSS 프레임워크
- TypeScript: 타입 안전성을 위한 언어

## 특징

```typescript
// 타입 안전한 블로그 포스트 처리
const post = allPosts.find((post) => post.slug === params.slug);
```

- MDX 지원으로 풍부한 콘텐츠 작성
- 🎨 아름다운 디자인과 반응형 레이아웃
- ⚡ 빠른 정적 사이트 생성
- SEO 최적화
