# Next.js 15 시작하기

Next.js 15가 출시되면서 많은 새로운 기능들이 추가되었다. 이번 포스트에서는 주요 변경사항과 시작하는 방법에 대해 알아보겠다.

## 주요 변경사항

### 1. Turbopack 안정화

```bash
# Turbopack을 기본으로 사용
npm run dev -- --turbopack
```

Turbopack이 안정화되어 개발 환경에서 더 빠른 번들링을 경험할 수 있다.

### 2. React 19 지원

- 서버 컴포넌트 개선: 더 나은 성능과 DX
- 새로운 Hooks: `use()` 훅 등 추가
- 향상된 Concurrent Features

### 3. App Router 개선

```typescript
// app/layout.tsx
export default function RootLayout({
  children,
}: {
  children: React.ReactNode
}) {
  return (
    <html lang="ko">
      <body>{children}</body>
    </html>
  )
}
```

## 시작하기

### 새 프로젝트 생성

```bash
npx create-next-app@latest my-app
cd my-app
npm run dev
```

### 기존 프로젝트 업그레이드

```bash
npm install next@latest react@latest react-dom@latest
```
