# React 학습 가이드 - 컴포넌트 Import/Export

## 컴포넌트 Import/Export

### 문제 상황

한 파일에 많은 컴포넌트를 선언하면 파일이 커지고 탐색이 어려워진다.

### 해결책: 파일 분리

파일 구조:

```
src/
  ├── Gallery.js      # Gallery 컴포넌트
  └── Profile.js      # Profile 컴포넌트
```

### Export/Import 예제

Profile 컴포넌트를 별도 파일로 분리하고, Gallery에서 불러와 사용하는 방법이다.

1단계: Profile.js 파일 생성

```jsx
// Profile.js
export default function Profile() {
  return <img src="https://i.imgur.com/MK3eW3As.jpg" alt="Katherine Johnson" />;
}
```

- `export default`를 앞에 붙이면 이 컴포넌트를 다른 파일에서 가져다 쓸 수 있다.
- 파일 하나에 하나의 주요 컴포넌트만 있을 때 default export를 사용한다.

2단계: Gallery.js에서 불러오기

```jsx
// Gallery.js
import Profile from "./Profile.js";

export default function Gallery() {
  return (
    <section>
      <h1>Amazing scientists</h1>
      <Profile />
      <Profile />
      <Profile />
    </section>
  );
}
```

- `import Profile from './Profile.js'`: Profile 컴포넌트를 불러온다.
- `'./Profile.js'`: 현재 폴더(`.`)에 있는 Profile.js 파일을 의미한다.
- 이제 `<Profile />`를 사용할 수 있다.

### Export 방식

React에서는 두 가지 Export 방식을 사용한다.

| 방식           | Export                                | Import                                 | 설명                        |
| -------------- | ------------------------------------- | -------------------------------------- | --------------------------- |
| Default Export | `export default function Button() {}` | `import Button from './Button.js'`     | 파일당 하나의 주요 컴포넌트 |
| Named Export   | `export function Button() {}`         | `import { Button } from './Button.js'` | 파일당 여러 컴포넌트        |

차이점:

1. Default Export:
   - 한 파일에 하나만 사용 가능
   - Import 시 중괄호 `{}` 없이 사용
   - Import 시 이름을 자유롭게 변경 가능: `import MyButton from './Button.js'`

2. Named Export:
   - 한 파일에 여러 개 사용 가능
   - Import 시 반드시 중괄호 `{}` 사용
   - 정확한 이름으로 Import 해야 함

권장 사항:

- 파일당 하나의 컴포넌트 → Default Export 사용 (대부분의 경우)
- 파일당 여러 컴포넌트 → Named Export 사용

### Vue와 비교

```html
<!-- Vue: Profile.vue -->
<template>
  <img src="..." alt="Katherine Johnson" />
</template>

<script setup>
  // 자동으로 export됨
</script>
```

```jsx
// React: Profile.js
export default function Profile() {
  return <img src="..." alt="Katherine Johnson" />;
}
```
