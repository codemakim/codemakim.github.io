## 0) 점(point) vs 벡터(vector) — 출발선 정리
- **점**: 좌표평면의 “위치”. 예) 집 주소처럼 (3, 2).  
- **벡터**: “어느 방향으로 얼마나 움직였는가”를 나타내는 **화살표**.  
  - 보통 원점 (0,0)에서 출발하는 화살표로 생각한다.  
  - 같은 위치의 점이라도, **어디서부터 움직였는지**가 벡터에선 중요.

> 기억하기: **점 = 위치**, **벡터 = 이동**.  
> 실무 감각: 그래픽스/게임/물리엔진/딥러닝 등에서 벡터는 “방향+크기”의 기본 단위다.

## 1) 벡터 표기법 — 어떻게 쓰고 읽나?

### 수식
$begin:math:display$
\\vec{v} = [x, y] \\quad \\text{(2D 벡터)}, \\qquad
\\vec{v} = [x, y, z] \\quad \\text{(3D 벡터)}
$end:math:display$

### 설명
- $begin:math:text$[x, y]$end:math:text$는 “오른쪽으로 x, 위로 y”만큼 이동한 화살표.  
- 예: $begin:math:text$\\vec{v} = [3, 4]$end:math:text$는 오른쪽 3, 위로 4 → 좌표평면에서 (0,0)→(3,4) 화살표.

### 코드
```python
import numpy as np
v2 = np.array([3, 4])       # 2차원 벡터
v3 = np.array([1, -2, 0.5]) # 3차원 벡터
print(v2.shape, v3.shape)   # (2,) (3,)
```

---

## 2) 벡터의 크기(길이, norm) — 피타고라스 정리 그대로

### 수식
$begin:math:display$
\\|\\vec{v}\\| = \\sqrt{x^2 + y^2} \\quad (2D), \\qquad
\\|\\vec{v}\\| = \\sqrt{x^2 + y^2 + z^2} \\quad (3D)
$end:math:display$

### 설명
- [밑변, 높이]로 직각삼각형을 만들면, 빗변이 바로 벡터의 길이.  
- 현실 감각: 속도(스피드)처럼 **얼마나 큰가**만 보고 싶을 때 크기를 쓴다.

### 코드
```python
import numpy as np
v = np.array([3, 4])
print(np.linalg.norm(v))  # 5.0 (sqrt(3^2 + 4^2))
```

---

## 3) 벡터의 덧셈 — 끝과 끝을 이으면 “한 번에 이동”과 같다

### 수식
$begin:math:display$
[a_x, a_y] + [b_x, b_y] = [a_x + b_x, \\; a_y + b_y]
$end:math:display$

### 설명
- 먼저 $begin:math:text$\\vec{a}$end:math:text$만큼 움직이고, 거기서 $begin:math:text$\\vec{b}$end:math:text$만큼 더 움직이면 = $begin:math:text$\\vec{a}+\\vec{b}$end:math:text$.  
- **교환법칙/결합법칙**이 성립한다: $begin:math:text$\\vec{a}+\\vec{b}=\\vec{b}+\\vec{a}$end:math:text$, $begin:math:text$(\\vec{a}+\\vec{b})+\\vec{c}=\\vec{a}+(\\vec{b}+\\vec{c})$end:math:text$

### 코드
```python
import numpy as np
a = np.array([3, 2])
b = np.array([1, 4])
print(a + b)  # [4 6]
```

## 4) 스칼라 곱 — 길이만 바꾸고, 방향은 유지(부호는 방향을 뒤집음)

### 수식
$begin:math:display$
k \\cdot [x,y] = [k x, \\; k y]
$end:math:display$

### 설명
- $begin:math:text$k>1$end:math:text$: 길이가 $begin:math:text$k$end:math:text$배로 늘어난다.  
- $begin:math:text$0<k<1$end:math:text$: 길이가 줄어든다(축소).  
- $begin:math:text$k=0$end:math:text$: 영(0)벡터가 되며, **방향이 사라진다**.  
- $begin:math:text$k<0$end:math:text$: 길이가 $begin:math:text$|k|$end:math:text$배가 되고 **방향이 반대로 뒤집힌다**.

### 코드
```python
import numpy as np
a = np.array([3, 2])
print(2 * a)   # [6 4]
print(-1 * a)  # [-3 -2] (방향 반전)
```

## 5) 정규화(normalization) — 방향만 보고 싶을 때 길이를 1로 만들기

### 수식
$begin:math:display$
\\hat{v} = \\frac{\\vec{v}}{\\|\\vec{v}\\|}
$end:math:display$

### 설명
- 정규화하면 “단위벡터”가 된다(길이=1).  
- 비교(유사도 계산 등)에서 **방향만** 중요할 때 자주 쓴다.  
- **주의**: $begin:math:text$\\vec{v}=\\vec{0}$end:math:text$인 경우는 나눌 수 없다(0으로 나누기).

### 코드
```python
import numpy as np
v = np.array([3, 4])
unit_v = v / np.linalg.norm(v)
print(unit_v)               # [0.6 0.8]
print(np.linalg.norm(unit_v))  # 1.0
```

## 6) (보너스) 방향각 — 벡터가 바라보는 각도

### 개념
- 2D에서 벡터가 x축과 이루는 각도는 `atan2(y, x)`로 구한다(라디안 단위).  
- $begin:math:text$\\text{deg} = \\text{rad} \\times \\frac{180}{\\pi}$end:math:text$

### 코드
```python
import numpy as np
v = np.array([3, 4])
angle_rad = np.arctan2(v[1], v[0])
angle_deg = np.degrees(angle_rad)
print(angle_rad, angle_deg)  # 0.927..., 53.13... (대략 3-4-5 삼각형 각도)
```

## 7) 종합 코드 예시 — 한 번에 확인

```python
import numpy as np

# 벡터 정의
a = np.array([3, 2])
b = np.array([1, 4])

# 크기
print("||a|| =", np.linalg.norm(a))
print("||b|| =", np.linalg.norm(b))

# 덧셈
print("a+b =", a + b)

# 스칼라 곱
print("2a =", 2*a)
print("-a =", -a)

# 정규화
unit_a = a / np.linalg.norm(a)
print("unit a =", unit_a, "||unit a|| =", np.linalg.norm(unit_a))
```

## 8) 오늘 핵심 요약
- 벡터 = **방향 + 크기**가 있는 화살표  
- 크기(norm) = 피타고라스 정리  
- 덧셈 = 성분별 더하기(이동을 이어붙임)  
- 스칼라 곱 = 길이 변화(+ 부호면 방향 반전)  
- 정규화 = 길이를 1로 만들어 방향만 남김(0벡터 주의)
