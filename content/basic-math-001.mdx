Day 1 — 벡터(Vector) 기초

1. 벡터란 무엇일까?
	•	네가 아는 **숫자(스칼라)**는 크기만 있음. 예: 5 → 길이가 5cm, 돈 5원, 사과 5개.
	•	벡터는 크기 + 방향을 동시에 가진 것.
	•	예: “오른쪽으로 3칸, 위로 2칸 이동” → 화살표로 표현할 수 있음.
	•	좌표평면에서 화살표(→) 로 생각하면 돼. 시작점은 보통 (0,0) 원점에서 출발한다고 생각.

⸻

2. 벡터 표기법

2차원 벡터는 이렇게 씀:
\vec{v} = [x, y]
예:
\vec{v} = [3, 4]
이건 원점에서 오른쪽으로 3, 위로 4 움직인 위치를 가리키는 화살표.

⸻

3. 벡터의 크기 (길이)

피타고라스 정리 기억하지?
삼각형 빗변 길이 = √(밑변² + 높이²).
벡터의 길이도 똑같아.

\|\vec{v}\| = \sqrt{x^2 + y^2}

예: \vec{v} = [3, 4] →
\|\vec{v}\| = \sqrt{3^2 + 4^2} = \sqrt{9+16} = \sqrt{25} = 5

즉, [3,4] 벡터의 길이는 5.

⸻

4. 벡터의 덧셈

벡터 두 개를 더하면, 단순히 x끼리, y끼리 더함.

[a_x, a_y] + [b_x, b_y] = [a_x+b_x, a_y+b_y]

예:
[3, 2] + [1, 4] = [3+1, 2+4] = [4, 6]

시각적으로는 화살표 두 개를 이어 붙이는 것과 같아.
즉, “먼저 a만큼 이동 → 이어서 b만큼 이동 = 한 번에 (a+b) 만큼 이동”.

⸻

5. 벡터의 스칼라 곱

숫자(스칼라)를 곱하면, 방향은 안 바뀌고 길이만 늘어나거나 줄어듦.

k \cdot [x,y] = [k\cdot x, k\cdot y]

예:
2 \cdot [3,2] = [6,4] → 같은 방향이지만 길이가 2배.

⸻

6. 벡터의 정규화 (단위 벡터)

“길이가 1인 벡터”로 바꾸는 것.
이유: 방향만 보고 싶을 때 유용.

\hat{v} = \frac{\vec{v}}{\|\vec{v}\|}

예: [3,4] → 길이가 5였으니,
\hat{v} = [3/5, 4/5] = [0.6, 0.8].
이 벡터 길이는 1.

⸻

7. 코드 예시 (NumPy)

import numpy as np

v = np.array([3,4])       # 벡터 [3,4]
norm_v = np.linalg.norm(v)  # 크기 계산
print("||v|| =", norm_v)    # 5.0

a = np.array([3,2])
b = np.array([1,4])
print("a+b =", a+b)         # [4 6]

scaled = 2*a
print("2a =", scaled)       # [6 4]

unit_v = v / norm_v
print("unit v =", unit_v)   # [0.6 0.8]


⸻

8. 정리
	•	벡터 = 크기와 방향을 가진 화살표
	•	크기(norm) = 피타고라스 정리
	•	덧셈 = 성분끼리 더하기 (끝-끝 연결)
	•	스칼라 곱 = 길이만 변함
	•	정규화 = 길이를 1로 맞춰 방향만 남김
